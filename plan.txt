AUTOTRADER FLAG SYSTEM (Session 5)
─────────────────────────────────────────

New DB table:
  position_flags (symbol PK, autotrader_on TINYINT, updated_at DATETIME)

Rules:
  1. Alpha autotrader buy → sets flag=1 for purchased symbol
  2. Manual buy (NEW position, not in portfolio) → sets flag=0
  3. Manual buy (EXISTING position) → flag unchanged
  4. Alpha autotrader exit/manage → only for positions where flag=1
  5. Phoenix autotrader → buys DISABLED; exits only for flag=1 positions
  6. Portfolio UI → ⚡ AT: ON (blue) / AT: OFF (grey) toggle per row
  7. Alpha buy → skips any symbol already in portfolio (any flag value)
  8. No Pick flag works independently of AT flag

Manual sell workflow:
  Set AT=OFF → sell desired qty → optionally re-enable AT=ON for remainder

─────────────────────────────────────────
LONG HAUL TAB CRITERIA
─────────────────────────────────────────

Filtered from stock_signals (watchlist stocks only):
  - div_yield > 0        (pays dividend)
  - pct_from_52high ≤ -20  (at least 20% below 52-week high)
  - pe_trailing < 35 OR pe_forward < 28  (not overvalued)
  - beta < 1.5           (fairly stable)
Sorted by dividend yield descending.
Columns: Symbol, Price/Chg, Div Yield, vs 52wk High, P/E, Fwd P/E, Beta, Signal, Sector
